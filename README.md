# Profit Mix Optimizer (1–3 Tracks)

אפליקציית Streamlit שמחשבת את השילובים המיטביים בין 1–3 מסלולים.

## קלטים נתמכים

1) **Excel רחב (כמו הקובץ שלך)** – בכל לשונית:
- שורה 1: `פרמטר | <גוף 1> | <גוף 2> | ...`
- בעמודה הראשונה יש מדדים (פרמטרים)
- האפליקציה מחלצת את הציון מהשורה הראשונה שמכילה `שארפ` ("מדד שארפ")

2) **CSV/Excel מנורמל** עם עמודות:
- `provider` – שם הגוף
- `track` – שם המסלול
- `score` – ציון מספרי

## הרצה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```
